Tools

Node.js is an open-source server environment that takes use of JavaScript and is able to run on multiple platforms (Windows, Linux, Mac OS X etc.).

Node.js allows for generation of dynamic page content, CRUD operations on files on the server and in the database and collect form data.
A Node.js file can contain tasks that will executed on certain events, typically someone trying to access a port on the server. It must be initiated on the server before having any effect.

To see how to run Node.js files, see the Terminal Command page.

Nodemon is a tool that is used to automatically restart node applications everytime a file changes in the directory are detected. It helps the developer with faster implementation.

Express is a free, open source minimal and flexible Node.js back end web application framework that is used to build web application and APIs.

For the applications we made we took use of Express' Routing, which refers to how an application's endpoints respond to client requests. The routing methods corresponds to the HTTP request methods. For example app.get() is used to handle GET requests and app.post() is used to handle POST requests.

NPM (Node Package Manager) is a Software Library which works as a Package Manager and Installer. It also includes a CLI (Command Line Client) which can be used to install software.

To see how NPM commands work in practice, see the Terminal Commands section.
