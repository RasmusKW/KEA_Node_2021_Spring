Terminal Commands

  • ls is used to list items in the current directory.
  • ls -a will list all files including hidden files.
  • ls -F gives a full listing, indicating what type files are by putting a slash after directories and a star after executable files.
  • ls -l gives a long listing of all files.

Used to navigate in directories (cd = Change Directory).
E.g: cd C:\Users\Computer | Tip: Use TAB to auto complete

node is used to run your node application
E.g: node app.js

nodemon is a tool that is used to automatically restart the node application everytime a change is made
making implementation faster
E.g: nodemon app.js

npm (Node Package Manager) is a Software Library which works as a Package Manager and Installer.
It also includes a CLI (Command Line Client) which can be used to install software.
E.g: npm install Express
